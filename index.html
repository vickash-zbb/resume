UI/UX Design + Web Development

Web Design

UX (User Experience) is about creating websites or apps with users in mind. This means making pretend people called "user personas," doing lots of research on real users, and making simple versions of the website to plan how it will work. I focus on understanding what users need to make sure the website or app is easy and helpful for them to use.

I study how users interact with things and design a special look for them that meets their needs and makes them happy. I focus on solving problems with smart design, which leads to a unique and user-friendly look.

Development

I make cool website designs using HTML, CSS, and JavaScript to keep everything looking and working smoothly for users. I mix my coding skills with design ideas to make websites that look great and work well

I focus on making websites that adapt to any device, like phones or tablets, so everyone has a great experience. It's important to me that the website works well on any device, making it easy for people to use no matter what they're using to access it

Template customization

I customize templates using HTML, CSS, and JavaScript to create modern and responsive designs. By changing default layouts, I make sure the website looks great and stays consistent with the brand. This gives users a smooth and enjoyable experience.

I'm skilled at making one-of-a-kind Bootstrap designs with Sass, a tool that makes styling easier. I combine customization with style effortlessly to create websites that look great and work well.

React
